1. PHP5基础

1.1 PHP简介

PHP（全称: Hypertext Preprocessor）是，是一种服务器端脚本语言，用于创建动态页面。
PHP+APACHE+MYSQL是目前最流行的WEB服务器应用开发组合：

  • PHP：服务器端脚本语言，用于生成动态页面。
  • APACHE：Apache服务器，是一款开源的HTTP服务器软件。
  • MYSQL：MySQL数据库，是一种关系型数据库管理系统。

2. 网站开发

因为PHP解析一个文件的时候，会寻找开始和结束标记<?php和?>

,所以可以将PHP嵌入到各种不同的文档中，而且在PHP中可以执行各种代码，可以实现网站的动态功能，如HTML标记<\br>.

注意，可以使用//注释，使用/**/进行多行注释，也可以使用#注释。

2.1 数据类型

PHP支持八种原始类型，其中有标量类型4种和复合类型2种，特殊类型2种。

  • 标量类型：
    1. string: 字符串类型，包括字符、数字、符号等。
    2. boolean: 布尔类型，true或false,大小写不敏感
    3. integer: 整型，包括正整数、负整数、0。
    4. double: 浮点型，包括小数。
  • 复合类型：
    1. array: 数组类型，可以存储多个值。
    2. object: 对象类型，可以存储多个属性和方法。
  • 特殊类型：
    1. resource: 资源类型，可以操作系统资源，如文件、数据库连接等。
    2. NULL: 空类型，表示一个空值。

可以使用gettype()函数获取变量的数据类型，如果要测试某种类型，使用is_xxx()函数，如is_string()、is_array()等:

2.1.1 布尔型

布尔型只有两个值：true和false，大小写不敏感
当转换成布尔值时，以下值会被认定为false：

  • Boolean类型false
  • 空字符串（””）和字符串’0’
  • 整型值0
  • 浮点型值0.0
  • 空数组（array()）
  • 空对象 没有单元的对象，仅适用于PHP4
  • NULL值
    其他值都会被认定为true。

2.1.2 整型

可以使用intval()函数将其他类型转换为整型：

  1. 布尔型：true转换为1，false转换为0。
  2. 浮点型：数字会被丢弃小数位，如果超出整数的范围，则不确定。
  3. 字符串：如果字符串包含有效的整数值，则转换为整数。

2.1.3 浮点型

  1. 永远不要比较两个浮点数是否相等，因为存在精度问题。
  2. 简单的十进制分数在转换为内部二进制格式情况下会change，导致精度丢失，floor（（0.1+0.7）*10）通常会返回7而不是8。

2.1.4 字符串

PHP里面，可以使用单引号，双引号和定界符来表示字符串。

  • 单引号：
  • 1.在字符串里面要表示单引号，使用反斜杠\转义（\）
  • 2.单引号不进行其他的转义,如\n
  • 3.单引号不进行变量的解析。
  • 双引号：
  • 1.双引号中出现的变量会被值替换。
  • 定界符：<<<
  • 1.在定界符后提供一个标识符，之后输入字符串，用同样的标识符结束。
  • 2.标识符不能被缩进，分号之前和之后都不能有空格，结束标识符前面要是换行符。
  • 3.不能用定界符语法初始化类成员。

2.1.5 序列化

序列化serialize（）：将PHP的值转换成字符串以永久地存储，如果建立了WDDX支持，还可以转换成XML结构。

2.1.6 字符串与算术

  1. 如果字符串中包括’.’,’E’(大小写不敏感)，会将其作为浮点数来求值，否则视为整数。
  2. 从字符串最前面合法的数字数据（可以由可选的正负号）开始，将其作为值，否则值为0.

什么叫做PHP将一个字符转换成整型时不能得到该字符的编码，需要使用ord（）和chr（）。

PHP 中的类型转换确实可能会导致一些意想不到的结果，特别是在将字符转换为整数时。在 PHP 中，如果尝试将一个字符（如 ‘A’）转换为整数，实际上得到的是该字符的 ASCII 值，比如 ‘A’ 的 ASCII 值是 65。
然而，问题可能出现在非 ASCII 字符上，特别是对于多字节字符或特殊符号。PHP 中的整数转换可能会产生不一致的结果，这是因为 PHP 的整数类型有其限制和特殊行为。对于一些特殊的字符或多字节字符，转换后的整数可能不是字符的预期编码值。
例如，对于 Unicode 字符或 UTF-8 编码的字符，PHP 可能无法正确地将其转换为整数，或者转换后的整数可能与该字符的 Unicode 码点不一致。这种情况下，就会出现“不能得到该字符的编码”的情况。

    10.QUERY_STRING:查询的字符串，即url第一个问号后的内容。
    11.DOCUMENT_ROOT:当前运行脚本所在的文档根目录。

2.7 包含代码

include()和require()函数用来包含外部文件，区别是：

  1. include()如果文件不存在，会产生一个警告，但脚本会继续执行，更适合html文件
  2. require()如果文件不存在，会产生一个致命错误，脚本终止执行，更适合包含函数库文件。

可以设置php.ini中的allow_url_include选项，来允许包含远程文件。
可以使用PHP内置函数get_included_files()来查看已经被包含的所有脚本————返回一个数组，包含每个脚本的完整路径。
