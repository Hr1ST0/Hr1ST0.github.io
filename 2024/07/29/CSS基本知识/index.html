类的创建

创建一个类有两步：

  1. 为HTML中的的元素增加一个class属性
  2. 在CSS中定义一个类选择器，并应用到相应的元素上
  3. 如果希望所有类中的元素都有同一种样式，可以这样写规则：
    4.一个元素要加入多个类，可以用空格分隔：
    <p class=”my-class my-other-class”>This is a div with multiple classes</p>

CSS 只能对<BODY>里面的元素增加样式,也可以对<BODY>进行指定，子元素会通过继承获得该样式。如果子元素中有一些不打算继承的样式，可以直接在子元素上设置。

对于img标签，CSS无法直接设置，但是可以设置img标签的父元素的样式，然后通过background-image属性来设置图片。

可以将<STYLE>中的样式提取出来，另外取名为“example.css”，放在HTML文件同级目录下。然后在<HEAD>中使用<LINK>标签引入样式文件：
<LINK type=”text/css” rel=”stylesheet” href=”example.css”>
rel属性的值为“stylesheet”表示引用样式表。
