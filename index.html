类的创建

创建一个类有两步：

  1. 为HTML中的的元素增加一个class属性
  2. 在CSS中定义一个类选择器，并应用到相应的元素上
  3. 如果希望所有类中的元素都有同一种样式，可以这样写规则：
    4.一个元素要加入多个类，可以用空格分隔：
    <p class=”my-class my-other-class”>This is a div with multiple classes</p>

CSS 只能对<BODY>里面的元素增加样式,也可以对<BODY>进行指定，子元素会通过继承获得该样式。如果子元素中有一些不打算继承的样式，可以直接在子元素上设置。

对于img标签，CSS无法直接设置，但是可以设置img标签的父元素的样式，然后通过background-image属性来设置图片。

可以将<STYLE>中的样式提取出来，另外取名为“example.css”，放在HTML文件同级目录下。然后在<HEAD>中使用<LINK>标签引入样式文件：
<LINK type=”text/css” rel=”stylesheet” href=”example.css”>
rel属性的值为“stylesheet”表示引用样式表。

HTML 基础知识

  1. 元素=开始标记+内容+结束标记
  2. 开始标记可以有属性：type
  3. 所有页面都要有标签
  4. 可以在 <STYLE>标签中定义CSS样式，<STYLE>标签要在<HEAD>元素里。
  5. 可以在<SCRIPT>标签中定义JavaScript脚本，<SCRIPT>标签要在<BODY>元素里。
  6. 字符实体：
    假如说在HTML中需要显示<和>符号，则需要使用字符实体，如果要显示字符实体，在字符实体前面加上\即可。

    &lt; &gt; &amp; &quot; &apos;
    &lt; 小于号
    &gt; 大于号
    &amp; 与号
    &quot; 双引号

7.注释：<!– 注释内容 –>

8.有序列表和无序列表：

有序列表：<ol> <li> </li> </ol>
无序列表：<ul> <li> </li> </ul>

9.文档类型定义<!DOCTYPE html>表明了现在的HTML标准

<META charset=”UTF-8”> 定义了文档的字符编码
ALT属性是<IMG>元素中的必要元素，当图像不能显示时，显示ALT属性中的文本。

引用标签

1.短引用<q>
2.长引用<blockquote>

域名和URL

域名具有唯一性，唯一标识网站。

  1. URL叫做统一资源定位符，是域名和资源之间的链接，它由协议、网站名和资源的一个绝对地址组成。
    例如，https://www.google.com/search?q=hello+world
  2. 协议：https表示安全超文本传输协议，http表示超文本传输协议。
  3. 网站名：www.google.com表示域名为google.com的www子域名。
  4. 资源：/search?q=hello+world表示搜索引擎搜索hello world。

index.html和default.htm是网站的默认首页，如果指定一个目录而没有指定文件名，则默认访问这两个页面。

a标签

1.想从一个页面链接到另一个页面，使用<A>标签。
2.<A>元素的href属性指定了链接的目标文件，<A>元素的内容是链接的显示文本，文字和图像都可以作为连接的目标，当我们点击之后就会跳转到href指定的页面。对于网站中的其他页面，使用相对路径，而对于外部链接才使用URL.
3.如果链接的目标是当前页面的某个位置，可以使用锚点（anchor）来指定目标位置，使用<A>标签的name属性指定锚点的名称，然后在href属性中使用#加上锚点的名称来指定链接的目标位置。
4.父文件夹用../表示，子文件夹用./表示。
5.可以用id属性在页面中创建一个目标，使用#后面加上目标名，可以直接链接到那个地方
