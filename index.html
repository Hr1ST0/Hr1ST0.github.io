4. 类与对象

模块被组织成了一个个拥有属性和方法的对象————

  1. 方法是对象所做的动作，作为函数。
  2. 属性是对象的特征，作为变量。

PHP5的对象模型是单重继承的，限制访问的，可以重载的，其将对象看成是与任何其他数据类型不同的类型，通过引用来传递。

4.1 对象的创建和使用：

类是一种类型，是实际变量的蓝图；而对象是类的实例，是实际变量，所以说，要发挥类的功能必须通过对象。

并非类所有的方法都可以被对象调用，只有public修饰的属性和方法才可以被外部访问，
private只能被类内部访问。

4.2 复制对象：

4.3 对象比较：

PHP5中，对象比较只有两个等号和三个等号：

  • 两个等号==： 两个对象是同一个类的实例，并且它们有相同的属性值。
  • 三个等号===： 两个对象是同一个类的实例，并且它们有相同的属性值和类型。

4.4 自动加载对象

PHP5中，可以使用__autoload()方法来实现自动加载对象。

修饰方法的关键词：final，

  • final修饰的方法不能被子类重写，只能被当前类重写。
  • final修饰的类不能被继承。

4.5 范围解析操作符(::)

范围解析操作符(::)用于在没有声明实例的情况下访问类中的函数或者基类中的函数和变量。

4.6 静态成员

静态成员是指在类中定义的变量和方法，不依赖于对象，可以被类自身调用，由所有类的实例共享。

  1. 静态属性
  2. 静态方法
    静态方法可以完全访问类的属性（不管限定语），但不存在$this指针。

4.7 继承

一个类可以继承另一个类，继承的类称为父类（基类），被继承的类称为子类（派类）。
派生类包含基类的所有属性和方法，可以加上其他属性和方法，也可以覆写基类的方法和属性。（只多不少）

4.8 抽象类和抽象方法

抽象方法在一个类中声明，在一个类中声明（没有代码），在该类的子类中实现，看作子类中一般方法的占位符（占地方但没有作用）。
如果类中存在抽象方法，那么该类必须被声明为抽象类，抽象类不能实例化，只能被继承后被实例化。（作用类似于模板）

4.9 接口（interface和implements）

接口是抽象类，只有抽象方法而没有普通方法。

4.10 构造函数（__construct）和析构函数（__destruct）

构造函数是类的构造函数，在创建对象时自动调用，用来对对象进行初始化。
析构函数是类的析构函数，在对象被销毁时自动调用，用来释放资源。

派生类的对象要执行基类的构造函数，必须在派生类的构造函数中显式调用：parent::__construct;

4.11 访问控制

private属性也可以被外部代码共享：定义了一对public方法（get和set），这样就可以提取出private属性的值。

一个子类可以通过覆写父类方法来改变方法的访问方式，这里有一些限制：

  • public只能改为public
  • protected可以改为public或者protected
  • private不能被修改
  • final关键字只能用于方法，该方法在子类中不会被覆写，但final不能用于属性。
  • php的绑定问题：
    在编程过程中调用一个父类和子类中都存在的方法或访问一个属性，PHP会先判断实例属于哪种对象类型，再调用相应的类中的方法和属性。

4.12 重载

4.12.1 属性重载

属性重载通过以下方法实现：

  1. __get()方法用于获取属性的值
  2. __set()方法用于设置属性的值。
  3. __isset()方法用于判断属性是否设置。
  4. __unset()方法用于删除属性。

在PHP中，魔术方法 __isset() 用于检查对象的属性是否已设置（即是否存在于对象中）。然而，对于公有属性（public properties），这些属性的访问是直接通过对象实例来进行的，而不会触发 __get() 或 __isset() 方法。
相比之下，当你访问一个不存在的动态属性（如 $obj->test），由于该属性并不存在，就会激活 __get() 和 __isset() 方法来处理这种情况。

4.12.2 方法重载(__call)

当你调用未定义方法，方法名和接收到的参数将会传给__call()方法。

4.13 序列化与反序列化serialize（）、unserialize（）

序列化是指将一个对象转换为一个字符串，反序列化是指将一个字符串转换为一个对象。
当一个对象被序列化，php会调用__sleep方法（如果存在），返回一个包含需要序列化的属性的数组。
被序列化的话，会调用__wakeup方法（如果存在），将属性恢复到对象中。二者都不接受参数的输入。

1. PHP5基础

1.1 PHP简介

PHP（全称: Hypertext Preprocessor）是，是一种服务器端脚本语言，用于创建动态页面。
PHP+APACHE+MYSQL是目前最流行的WEB服务器应用开发组合：

  • PHP：服务器端脚本语言，用于生成动态页面。
  • APACHE：Apache服务器，是一款开源的HTTP服务器软件。
  • MYSQL：MySQL数据库，是一种关系型数据库管理系统。

2. 网站开发

因为PHP解析一个文件的时候，会寻找开始和结束标记<?php和?>

,所以可以将PHP嵌入到各种不同的文档中，而且在PHP中可以执行各种代码，可以实现网站的动态功能，如HTML标记<\br>.

注意，可以使用//注释，使用/**/进行多行注释，也可以使用#注释。

2.1 数据类型

PHP支持八种原始类型，其中有标量类型4种和复合类型2种，特殊类型2种。

  • 标量类型：
    1. string: 字符串类型，包括字符、数字、符号等。
    2. boolean: 布尔类型，true或false,大小写不敏感
    3. integer: 整型，包括正整数、负整数、0。
    4. double: 浮点型，包括小数。
  • 复合类型：
    1. array: 数组类型，可以存储多个值。
    2. object: 对象类型，可以存储多个属性和方法。
  • 特殊类型：
    1. resource: 资源类型，可以操作系统资源，如文件、数据库连接等。
    2. NULL: 空类型，表示一个空值。

可以使用gettype()函数获取变量的数据类型，如果要测试某种类型，使用is_xxx()函数，如is_string()、is_array()等:

2.1.1 布尔型

布尔型只有两个值：true和false，大小写不敏感
当转换成布尔值时，以下值会被认定为false：

  • Boolean类型false
  • 空字符串（””）和字符串’0’
  • 整型值0
  • 浮点型值0.0
  • 空数组（array()）
  • 空对象 没有单元的对象，仅适用于PHP4
  • NULL值
    其他值都会被认定为true。

2.1.2 整型

可以使用intval()函数将其他类型转换为整型：

  1. 布尔型：true转换为1，false转换为0。
  2. 浮点型：数字会被丢弃小数位，如果超出整数的范围，则不确定。
  3. 字符串：如果字符串包含有效的整数值，则转换为整数。

2.1.3 浮点型

  1. 永远不要比较两个浮点数是否相等，因为存在精度问题。
  2. 简单的十进制分数在转换为内部二进制格式情况下会change，导致精度丢失，floor（（0.1+0.7）*10）通常会返回7而不是8。

2.1.4 字符串

PHP里面，可以使用单引号，双引号和定界符来表示字符串。

  • 1.在字符串里面要表示单引号，使用反斜杠\转义（\）
  • 2.单引号不进行其他的转义,如\n
  • 3.单引号不进行变量的解析。
  • 双引号：
  • 1.双引号中出现的变量会被值替换。

2.1.6 字符串与算术

  1. 如果字符串中包括’.’,’E’(大小写不敏感)，会将其作为浮点数来求值，否则视为整数。
  2. 从字符串最前面合法的数字数据（可以由可选的正负号）开始，将其作为值，否则值为0.

什么叫做PHP将一个字符转换成整型时不能得到该字符的编码，需要使用ord（）和chr（）。

PHP 中的类型转换确实可能会导致一些意想不到的结果，特别是在将字符转换为整数时。在 PHP 中，如果尝试将一个字符（如 ‘A’）转换为整数，实际上得到的是该字符的 ASCII 值，比如 ‘A’ 的 ASCII 值是 65。
然而，问题可能出现在非 ASCII 字符上，特别是对于多字节字符或特殊符号。PHP 中的整数转换可能会产生不一致的结果，这是因为 PHP 的整数类型有其限制和特殊行为。对于一些特殊的字符或多字节字符，转换后的整数可能不是字符的预期编码值。
例如，对于 Unicode 字符或 UTF-8 编码的字符，PHP 可能无法正确地将其转换为整数，或者转换后的整数可能与该字符的 Unicode 码点不一致。这种情况下，就会出现“不能得到该字符的编码”的情况。

  6. 数组与任何其他类型比较，数组更大
  7. 对象和其他类型比较，对象更大（不能比较数组和对象）

域名和URL

域名具有唯一性，唯一标识网站。

  1. URL叫做统一资源定位符，是域名和资源之间的链接，它由协议、网站名和资源的一个绝对地址组成。
    例如，https://www.google.com/search?q=hello+world
  2. 协议：https表示安全超文本传输协议，http表示超文本传输协议。
  3. 网站名：www.google.com表示域名为google.com的www子域名。
  4. 资源：/search?q=hello+world表示搜索引擎搜索hello world。

index.html和default.htm是网站的默认首页，如果指定一个目录而没有指定文件名，则默认访问这两个页面。

a标签

1.想从一个页面链接到另一个页面，使用<A>标签。
2.<A>元素的href属性指定了链接的目标文件，<A>元素的内容是链接的显示文本，文字和图像都可以作为连接的目标，当我们点击之后就会跳转到href指定的页面。对于网站中的其他页面，使用相对路径，而对于外部链接才使用URL.
3.如果链接的目标是当前页面的某个位置，可以使用锚点（anchor）来指定目标位置，使用<A>标签的name属性指定锚点的名称，然后在href属性中使用#加上锚点的名称来指定链接的目标位置。
4.父文件夹用../表示，子文件夹用./表示。
5.可以用id属性在页面中创建一个目标，使用#后面加上目标名，可以直接链接到那个地方
